# Acerca de m-POSw

**m-POSw** (Mini POS Web) es un sistema de punto de venta diseñado para jornadas y eventos.

## Versión
Versión 1.0.0

## Desarrollo
Este programa fue confeccionado íntegramente con IA (Codex).

## Características principales
- **POS táctil**: Interfaz intuitiva para tablets y celulares
- **Múltiples métodos de pago**: Efectivo, Mercado Pago QR y Transferencias
- **Gestión de productos**: Productos simples, compuestos y materia prima
- **Control de stock**: Seguimiento en tiempo real del inventario
- **Cierre de caja**: Desglose por método de pago
- **Reportes**: Exportación a Excel con filtros por fecha
- **Estadísticas**: Gráficos de ventas y análisis

## Tecnologías
- **Frontend**: React + Vite + TypeScript
- **Backend**: NestJS + Prisma
- **Base de datos**: PostgreSQL
- **Infraestructura**: Docker Compose
