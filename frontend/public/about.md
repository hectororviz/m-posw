# Acerca de m-POSw

**m-POSw** (Mini POS Web) es un sistema de punto de venta diseñado para pequeños comercios, bufet o kioscos.

## Versión
Versión 1.0.0

## Desarrollo
Este programa fue confeccionado íntegramente con IA (Codex y OpenCode).

## Características principales
- **POS táctil**: Interfaz intuitiva para tablets y celulares. Permite mostrar u ocultar teclado integrado en la interfaz para el uso solo con pantallas.
- **Múltiples métodos de pago**: Efectivo, Mercado Pago QR y Transferencias(MercadoPago) y confirmacion de recepcion de pagos electrónicos.
- **Gestión de productos**: Productos simples, compuestos y materia prima para la creacion de recetas.
- **Control de stock**: Seguimiento en tiempo real del inventario de productos simples y materias primas.
- **Cierre de caja**: Desglose por método de pago y crecion de movimientos entradas/salidas de dinero.
- **Reportes**: Reportes en tiempo real y exportación a Excel con filtros por fecha.
- **Estadísticas**: Gráficos de ventas y análisis.

## Tecnologías
- **Frontend**: React + Vite + TypeScript
- **Backend**: NestJS + Prisma
- **Base de datos**: PostgreSQL
- **Infraestructura**: Docker Compose
